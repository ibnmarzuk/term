# 🌌 Core OS (APEX OS Workspace)

Core OS (officially integrated as APEX OS) is a high-performance, system-level workspace and developer ecosystem designed to manage, visualize, and orchestrate specialized autonomous intelligence agents. Built with a futuristic, cybernetically secure space-tech aesthetic, the platform equips systems engineers and product coordinators with the tools to trigger parallel execution sandboxes, monitor performance bottlenecks, map complex cognitive relationships, and manage generated software and documentation assets.

---

## 🚀 Key Architectural Pillars

1. **Autonomous Parallel Pipeline**: Move from a single input (bounty, PDF, workspace url) to a complete verified intelligence package through synchronized multi-stage agent workflows (Ingest, Research, Strategy, Engineering, Audit).
2. **Cognitive Telemetry & Real-Time Monitoring**: Observe live telemetry data via sub-millisecond line charts tracking resource utilization, pipeline latency, and active units.
3. **Interactive Sandboxing**: Explore sandbox sandpits where users can test designs, inspect compile states, and manage type-safe JSON and TypeScript artifacts.
4. **Agile Workflow Orchestration**: Dynamic drag-and-drop task tracking, allowing team leaders to manually balance agent operations and isolate bottlenecks.

## 📑 Feature & Page Walkthrough

### 1. **Public Marketing & Operational Suite**
*   **Landing Page (`/`)**: A dark-mode, high-contrast display layout introducing APEX OS's core proposition. Anchored by responsive micro-interactions, custom styling margins, and centered typographic pairings.
*   **About (`/about`)**: Chronicles the core movement of Switzerland/San Francisco engineers pushing isolated runtime architectures. Features the interactive **Join the Core OS Movement** centered call-to-action.
*   **Features (`/features`)**: Highlights technical architectures including high-performance telemetry, decentralized storage caches, and decoupled orchestration enclaves.
*   **Pricing (`/pricing`)**: A clean, transparent cost breakdown for individual researchers up to large sovereign enterprise teams.
*   **Contact (`/contact`)**: Gateway to coordinate directly with systems designers.
*   **Privacy & Terms (`/privacy`, `/terms`)**: Legally compliant, beautifully formatted administrative policies.
*   **Authentication Portal (`/login`, `/signup`)**: Minimalist inputs styled with high-contrast active highlights, connecting to the secure `AuthService` state manager.

---

### 2. **Telemetry & Orchestration Command Centers** (Protected Dashboard)

#### 🕹️ **Command Center (`/command-center`)**
*   **Real-Time Performance Monitor**: Features dynamic live charts mapping **Active Agent Units** (Research, Engineering, QA, etc.), **Pipeline Latency (ms)**, and **Resource Utilization (%)** updated continuously.
*   **One-Input Pipeline**: An interactive 4-stage wizard demonstrating structural inputs progressing through APEX's modular, automated execution.

#### 📁 **Agent Directory (`/agents`)**
*   A directory of specialized autonomous intelligence engines ready to deploy:
    *   *Research Analysts* executing market-sizing and competitive profiling.
    *   *Product Designers* mapping UX and interaction components.
    *   *Software Engineers* deploying full-stack boilerplates and Anchor programs.
    *   *Technical & Content Writers* drafting setup guides and tweet threads.
    *   *Growth & BizDev Strategists* formulating customer-acquisition projects.

#### 🛠️ **Project Workspace (`/workspace`)**
*   **Execution Graph**: Provides visual feedback of active, waiting, and completed pipeline agents.
*   **Interactive Controls**: Initiate and stop parallel enclaves, toggle deployment variables, and review output checklists.

#### 🛰️ **Opportunity Radar (`/radar`)**
*   An AI-driven market telemetry feed mapping global challenges, hackathons, and research grants.
*   Includes **Command Palette Search (CMD+K)** inputs, signal probability indices, and fit scores.

#### 🧠 **Intelligence Brain (`/brain`)**
*   An interactive digital canvas visualizing core cognitive mapping.
*   Generates a network of nodes (**Strategic Council**, **Router Engine**, **Knowledge Graph**) with real-time vector lines displaying current path tracking and exchange metrics.

#### 🧪 **Artifact Studio (`/artifacts`)**
*   A localized file explorer and generator to review, copy, and export output assets (`core-router.ts`, `database-schema.sql`, `design-tokens.json`).
*   Equipped with granular version histories, size flags, and author agent tracking.

#### 📋 **Tracking Board (`/tracking-board`)**
*   A stateful, interactive Kanban dashboard.
*   Utilizes **drag-and-drop operations** from `@dnd-kit/core` to move tasks smoothly between operational pipelines (**Completed**, **In Progress**, **Blocked**, **Pending**).

## 🛠️ Technology Stack & Dependencies

*   **Runtime Framework**: React 18+ powered by **Vite** and styled using **TypeScript**.
*   **Styling & Theming**: **Tailwind CSS** implementing a bespoke, midnight-green cybernetic theme.
*   **Animations & Transitions**: `motion/react` implementing smooth, context-preserving layout transitions.
*   **Analytical Visuals**: `recharts` mapping live performance performance indicators.
*   **Interactive Drag-and-Drop**: `@dnd-kit/core` and `@dnd-kit/sortable` powers state preservation across Kanban operations.
*   **Icons**: Clean vector indicators imported from `lucide-react`.

## 💻 Setup & Development

To get Core OS running in your local sandbox:

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Launch Dev Server**:
    ```bash
    npm run dev
    ```
3.  **Compile & Build Application**:
    ```bash
    npm run build
    ```
4.  **Run Linter**:
    ```bash
    npm run lint
    ```
